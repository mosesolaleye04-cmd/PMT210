# PMT 210 / ECN 214 — CBT Practice Exam

A Computer-Based Test (CBT) practice exam for **Principles of Economics** at FUTA. This is an interactive, self-contained HTML application for self-practice and learning.

## Features

- **Multiple Question Types**: Multiple choice, true/false, and fill-in-the-blank questions
- **Instant Feedback**: See correct answers and explanations immediately after each answer
- **Progress Tracking**: Real-time score updates as you work through the test
- **Flexible Sections**: Choose which sections to include in your practice session
- **Question Shuffling**: Optional randomization of question order
- **Results Review**: Complete summary with performance analysis and question-by-question review
- **Responsive Design**: Works on desktop, tablet, and mobile devices

## How to Use

### Online (GitHub Pages)
[**Open the CBT Practice Exam**](https://mosesolaleye04-cmd.github.io/PMT210/PMT210_ECN214_CBT.html)

### Local
1. Clone or download this repository
2. Open `PMT210_ECN214_CBT.html` in any modern web browser
3. No server or internet connection required (except to load fonts)

## How It Works

1. **Setup**: Select which question sections to include in your session
2. **Practice**: Answer questions and get immediate feedback
3. **Review**: See your final score and review all questions
